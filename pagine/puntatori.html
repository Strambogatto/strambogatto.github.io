📌 Guida Completa ai Puntatori in C

Scopri il potere dei puntatori nella programmazione C

Cosa sono i Puntatori?

Un puntatore è una variabile che memorizza l'indirizzo di memoria di un'altra variabile. È uno dei concetti fondamentali del C e permette di manipolare la memoria direttamente.

Perché sono importanti?

I puntatori sono la base di molte cose in C: passare parametri per modificarli dentro funzioni, creare strutture dati dinamiche (liste, alberi, grafi), lavorare con la memoria in modo flessibile. Senza puntatori, il C non sarebbe così potente. Probabilmente pensi "ma perché mi serve?" - bene, immagina di voler scrivere una funzione che modifichi una variabile passata come parametro. In molti linguaggi è semplice, ma in C devi usare un puntatore per farlo. Oppure immagina di voler creare un array ma non sai quanto grande servirà fino a runtime - qui entrano in gioco i puntatori e la memoria dinamica!

🎯 Operatori Fondamentali: & e *

I due operatori che devi capire per primi

L'operatore & (indirizzo)

L'operatore & significa "dammi l'indirizzo di memoria di questa variabile". Pensa alla memoria del computer come a una lunga serie di scatole numerate. Ogni variabile occupa una o più scatole. L'operatore & ti dice quale numero ha la prima scatola di quella variabile.

int x = 5;
int *ptr = &x; // ptr contiene l'indirizzo di x

Se x si trova alla scatola numero 1000, allora &x ritorna 1000. Questo numero può essere immagazzinato in un puntatore.

L'operatore * (dereferenziazione)

L'operatore * fa l'opposto: dato un puntatore (un indirizzo), ti ritorna il valore contenuto a quell'indirizzo. È come dire "vai alla scatola numero 1000 e dimmi che valore contiene".

int x = 5;
int *ptr = &x;
printf("%d", *ptr); // Stampa 5, il valore di x

Attenzione: * ha significati diversi in contesti diversi! In "int *ptr" significa "dichiarazione di puntatore", mentre in "*ptr = 10" significa "dereferenzia ptr e assegna 10 al valore puntato".

🔗 Dichiarazione e Inizializzazione

Come creare e usare i puntatori correttamente

Dichiarare un puntatore

Quando dichiari un puntatore, devi specificare il tipo di variabile che il puntatore punterà. Un puntatore a int è diverso da un puntatore a char.

int *ptr_intero; // Puntatore a int
char *ptr_carattere; // Puntatore a char
double *ptr_decimale; // Puntatore a double

Il tipo determina come il compilatore interpreta i dati e quanti byte saltare quando fai aritmetica sui puntatori (che vedremo dopo).

Inizializzare correttamente

IMPORTANTE: devi sempre inizializzare un puntatore prima di usarlo! Se non lo fai, il puntatore contiene un indirizzo casuale (un puntatore "selvatico") e dereferenziarlo causa un crash.

int x = 42;
int *ptr = &x; // Corretto: ptr punta a x

int *ptr; // SBAGLIATO: puntatore non inizializzato!
// *ptr = 5; // CRASH! ptr contiene spazzatura

Una buona pratica è inizializzare a NULL se non sai ancora a cosa dovrebbe puntare:

int *ptr = NULL; // Sicuro, poi lo assegniamo dopo

🎪 Puntatori a Puntatori

Quando un puntatore punta a un altro puntatore

Cos'è un puntatore a puntatore?

Se un puntatore contiene l'indirizzo di una variabile, allora un puntatore a puntatore contiene l'indirizzo di un puntatore! È come avere una catena di indirezioni.

int x = 42;
int *ptr = &x; // ptr punta a x
int **pptr = &ptr; // pptr punta a ptr

Come accedere ai valori

Per accedere al valore originale attraverso un puntatore a puntatore, usi * due volte:

printf("%d", *ptr); // Stampa 42 (il valore di x)
printf("%d", **pptr); // Stampa 42 (il valore di x attraverso ptr)
printf("%p", ptr); // Stampa l'indirizzo di x
printf("%p", pptr); // Stampa l'indirizzo di ptr

Quando serve?

I puntatori a puntatori sono utili quando vuoi modificare un puntatore all'interno di una funzione. Se passi solo un puntatore, modifiche solo cosa punta, non il puntatore stesso. Ma se passi un puntatore a puntatore, puoi cambiare il puntatore stesso.

void cambia(int **pp, int valore) {
  *pp = &valore; // Cambia cosa punta pp
}

📊 Puntatori e Array

Una relazione fondamentale che non immaginavi

Array decay: gli array diventano puntatori

Ecco un segreto che molti non sanno: in C, il nome di un array (senza le parentesi quadre) è automaticamente convertito in un puntatore al primo elemento! Questo è chiamato "array decay".

int arr[5] = {1, 2, 3, 4, 5};
int *ptr = arr; // arr si trasforma in puntatore al primo elemento
// È equivalente a: int *ptr = &arr[0];

Accedere agli elementi

Sia con la notazione array che con la notazione puntatore, accedi agli stessi elementi:

int arr[5] = {1, 2, 3, 4, 5};
int *ptr = arr;

printf("%d", arr[2]); // Stampa 3
printf("%d", ptr[2]); // Stampa 3
printf("%d", *(ptr + 2)); // Stampa 3
printf("%d", *(arr + 2)); // Stampa 3

Tutte e quattro sono equivalenti! In realtà, arr[i] è semplicemente zucchero sintattico per *(arr + i).

Perché è importante?

Questo è il motivo per cui gli array si comportano in un certo modo quando li passi alle funzioni. Quando passi un array a una funzione, in realtà passi un puntatore, non l'intero array! Per questo il compilatore non sa la dimensione dell'array dentro la funzione e tu devi passarla separatamente come parametro.

➕ Aritmetica dei Puntatori

Come navigare la memoria con i puntatori

Incremento e decremento

Puoi incrementare e decrementare puntatori come normali variabili, ma c'è una differenza: il computer automaticamente sposta di "una unità del tipo", non di un byte!

int arr[5] = {10, 20, 30, 40, 50};
int *ptr = arr; // ptr punta a arr[0] (indirizzo 1000)
ptr++; // ptr ora punta a arr[1] (indirizzo 1004, cioè 1000 + 4)
ptr++; // ptr ora punta a arr[2] (indirizzo 1008, cioè 1004 + 4)
printf("%d", *ptr); // Stampa 30

Nota: ogni int è 4 byte, quindi quando incrementi di 1, in realtà avanzi di 4 byte! Se fosse un puntatore a char (1 byte), avanzerebbe di 1 byte.

Addizione e sottrazione

Puoi aggiungere numeri ai puntatori per saltare multipli di elementi:

int arr[5] = {10, 20, 30, 40, 50};
int *ptr = arr;
printf("%d", *(ptr + 3)); // Stampa 40 (arr[3])
printf("%d", *(ptr + 4)); // Stampa 50 (arr[4])

ptr = ptr + 2; // ptr ora punta a arr[2]
printf("%d", *ptr); // Stampa 30

Differenza tra puntatori

Puoi anche sottrarre due puntatori per sapere quanti elementi distano:

int arr[5] = {10, 20, 30, 40, 50};
int *ptr1 = arr;
int *ptr2 = arr + 3;
printf("%d", ptr2 - ptr1); // Stampa 3 (distanza di 3 elementi)

⚠️ Il Vero Motivo dei Puntatori

Casi d'uso pratici che ti convinceranno

Caso 1: Modificare una variabile in una funzione

Senza puntatori, i parametri di funzione sono copie. Se modifichi il parametro, non modifichi l'originale:

void incrementa(int x) {
  x++; // Incrementa la COPIA, non l'originale
}

int main() {
  int num = 5;
  incrementa(num);
  printf("%d", num); // Stampa 5, non 6!
}

Con i puntatori, puoi modificare l'originale:

void incrementa(int *x) {
  (*x)++; // Incrementa il valore puntato
}

int main() {
  int num = 5;
  incrementa(&num);
  printf("%d", num); // Stampa 6!
}

Caso 2: Ritornare più di un valore

Una funzione non può ritornare più di un valore direttamente, ma con puntatori puoi modificare più variabili:

void dividi(int a, int b, int *quoziente, int *resto) {
  *quoziente = a / b;
  *resto = a % b;
}

int main() {
  int q, r;
  dividi(17, 5, &q, &r);
  printf("Quoziente: %d, Resto: %d", q, r);
}

Caso 3: Evitare copie di dati grandi

Se vuoi passare un array grande a una funzione, non vuoi copiarlo tutto in memoria. Passi solo un puntatore al primo elemento, che è molto veloce!
